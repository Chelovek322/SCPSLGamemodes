Presidential Escort
======
made by mkrzy
## Description
A Scientist (VIP) spawns with several NTF in the cell. They need escort the VIP to escape from the facility. 
There are also some SCPs in the facility. If the VIP dies, SCPs win. If the VIP escapes, humans win.
